JavaScript — Beginner to Intermediate Tutorial

1. Introduction

JavaScript is a lightweight, interpreted programming language primarily used to make web pages interactive.

  • Originally created in 1995 by Brendan Eich.
  • Runs in the browser and on servers (Node.js).
  • Core technologies of the web: HTML, CSS, and JavaScript.

2. Setup & Environment

                    # Run in browser console:
                    console.log("Hello JavaScript");

                    # Or install Node.js:
                    https://nodejs.org

                    # Run in terminal:
                    node script.js

Use editors like VS Code or WebStorm for best experience.

3. Basic Syntax

                    console.log("Hello, JS!");
                    // Single-line comment
                    /*
                      Multi-line comment
                    */
  • Statements end with semicolons (optional but recommended).
  • Case-sensitive and dynamically typed.

4. Variables & Data Types

                    let age = 25;           // block-scoped variable
                    const name = "Alice";   // constant
                    var city = "Paris";     // function-scoped

                    // Data types:
                    string, number, boolean, object, undefined, null, symbol, bigint

Use let and const instead of var for modern code.

5. Operators

                    // Arithmetic: +, -, *, /, %, **
                    // Comparison: ==, ===, !=, !==, <,>, <=, >=
                    // Logical: &&, ||, !
                    // Assignment: =, +=, -=, *=, /=
                    // Ternary: condition ? value1 : value2

Use === for strict equality (checks both type and value).

6. Control Flow

                    if (x > 0) {
                      console.log("Positive");
                    } else if (x === 0) {
                      console.log("Zero");
                    } else {
                      console.log("Negative");
                    }

                    for (let i = 0; i < 5; i++) console.log(i);

                    while (condition) {
                      // loop body
                    }

7. Functions

                    // Traditional
                    function greet(name) {
                      return "Hello " + name;
                    }

                    // Arrow function
                    const add = (a, b) => a + b;

                    console.log(greet("Alice"));
                    console.log(add(5, 3));

Arrow functions are concise and lexically bind this.

8. Objects & Arrays

                    // Object
                    const person = {
                      name: "Alice",
                      age: 25,
                      greet() {
                        console.log("Hi, I'm " + this.name);
                      }
                    };
                    person.greet();

                    // Array
                    const fruits = ["Apple", "Banana", "Cherry"];
                    fruits.push("Orange");
                    fruits.forEach(f => console.log(f));

9. DOM Manipulation

                    document.getElementById("demo").innerText = "Updated!";
                    document.querySelector(".box").style.color = "cyan";
                    document.createElement("div");

JavaScript can dynamically update HTML and CSS in real time.

10. Async JavaScript

                    // Callback
                    setTimeout(() => console.log("Done"), 1000);

                    // Promise
                    fetch("https://api.example.com/data")
                      .then(res => res.json())
                      .then(data => console.log(data))
                      .catch(err => console.error(err));

                    // Async/Await
                    async function getData() {
                      try {
                        const res = await fetch("https://api.example.com");
                        const data = await res.json();
                        console.log(data);
                      } catch (err) {
                        console.error(err);
                      }
                    }

Async functions simplify working with promises and asynchronous code.

11. Best Practices

  • Always use const or let instead of var.
  • Use === for equality checks.
  • Keep functions small and focused.
  • Use modern ES6+ features (arrow functions, destructuring).
  • Handle async errors with try...catch.

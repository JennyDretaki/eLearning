Git and GitHub — Beginner Tutorial

1. What is Git and GitHub

Git is a version control system that tracks changes in your code. GitHub is a platform that hosts Git repositories and allows collaboration between developers.

In short, Git keeps track of your changes locally, and GitHub lets you share them with others.

2. Setting up the Environment

Install Git on your machine:

                    # Windows
                    Download from https://git-scm.com

                    # macOS (Homebrew)
                    brew install git

                    # Linux (Ubuntu)
                    sudo apt install git

Verify the installation:

git --version

3. Configuring Git

Set your name and email for commits:

                    git config --global user.name "Your Name"
                    git config --global user.email "you@example.com"

4. Creating a Repository

Create a repository on GitHub and clone it locally:

                    git clone https://github.com/username/repository.git
                    cd repository

Or initialize a new local repository:

git init

5. Basic Git Workflow

Add and commit files:

                    git add .
                    git commit -m "Initial commit"

Push to GitHub:

git push origin main

6. Branching

Create and switch branches:

                    git branch feature-branch
                    git checkout feature-branch

Merge back into main:

                    git checkout main
                    git merge feature-branch

7. Pull Requests

After pushing your branch, go to GitHub and click “Compare & pull request”. Add a description, submit it, and once approved, merge it into main.

8. Common Git Commands

                    git status   # Check changes
                    git log      # View commit history
                    git diff     # Compare changes
                    git pull     # Fetch latest updates from remote

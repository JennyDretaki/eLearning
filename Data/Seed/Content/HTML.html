HTML — Beginner to Intermediate Tutorial

1. Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It provides structure and content for the web.

  • Defines headings, paragraphs, links, images, lists, and more.
  • Works with CSS and JavaScript to style and add functionality.
  • HTML5 is the latest version with semantic and multimedia support.

2. Basic Structure

                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <title>Page Title</title>
                    </head>
                    <body>
                        <h1>Hello, World!</h1>
                    </body>
                    </html>

3. Common Elements

                    <p>Paragraph</p>
                    <h1>Heading 1</h1>
                    <h2>Heading 2</h2>
                    <div>Division</div>
                    <span>Inline</span>

4. Attributes

                    <a href="https://example.com">Link</a>
                    <img src="image.jpg" alt="Description">
                    <input type="text" placeholder="Enter name">

Attributes provide additional information about elements.

5. Links & Images

                    <a href="https://openai.com">OpenAI</a>
                    <img src="logo.png" alt="OpenAI Logo">

6. Lists & Tables

                    <ul>
                      <li>Item 1</li>
                      <li>Item 2</li>
                    </ul>

                    <table>
                      <tr><th>Name</th></tr>
                      <tr><td>Alice</td></tr>
                    </table>

7. Forms

                    <form action="/submit" method="post">
                      <input type="text" name="username">
                      <input type="submit" value="Submit">
                    </form>

8. Semantic HTML

Semantic tags give meaning to content:

                    <header>Site Header</header>
                    <nav>Navigation</nav>
                    <main>Main Content</main>
                    <footer>Site Footer</footer>

9. Multimedia

                <video src="video.mp4" controls></video>
                <audio src="audio.mp3" controls></audio>
                <iframe src="https://example.com"></iframe>

10. Best Practices

  • Use semantic HTML for better accessibility.
  • Keep code clean and well-indented.
  • Always provide alt text for images.
  • Validate HTML using W3C validator.
  • Use comments to describe sections of code.

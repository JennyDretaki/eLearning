C++ — Beginner to Intermediate Tutorial

1. Introduction to C++

C++ is a high-performance general-purpose programming language. It supports procedural, object-oriented, and generic programming.

  • Created by Bjarne Stroustrup in 1980s.
  • Used for systems, games, embedded, and high-performance applications.
  • Compiled language producing fast executables.

2. Setup & Tooling

Install a compiler and editor:

                    // Windows: Visual Studio
                    // Linux: sudo apt install g++
                    // macOS: brew install gcc

Compile and run:

g++ hello.cpp -o hello
                ./hello

7. Object-Oriented Programming

                    class Animal {
                    public:
                      virtual void speak() { std::cout << "..." << std::endl; }
                      virtual ~Animal() = default;
                    };

                    class Dog : public Animal {
                    public:
                      void speak() override { std::cout << "Woof!" << std::endl; }
                    };

8. Pointers & Memory

                    int x = 10;
                    int* ptr = &x; // pointer
                    int& ref = x;    // reference

                    *ptr = 20; // modifies x
                    ref = 30;  // modifies x

                    int* arr = new int[5];
                    delete[] arr;

11. Modern C++

                    #include <memory>
                    std::unique_ptr<int> p = std::make_unique<int>(42);

12. Concurrency & Threads

                    #include <thread> <mutex> <iostream>
                    std::mutex m;
                    void worker(int id) {
                      std::lock_guard<std::mutex> lock(m);
                      std::cout << "Worker " << id << "\n";
                    }
                    int main() {
                      std::thread t1(worker,1);
                      std::thread t2(worker,2);
                      t1.join(); t2.join();
                    }

13. Best Practices

  • Use RAII and smart pointers
  • Prefer std::vector over raw arrays
  • Write small, reusable functions
  • Use const and references appropriately
  • Learn modern C++ features (C++11/14/17/20)

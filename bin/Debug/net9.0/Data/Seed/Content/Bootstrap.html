Bootstrap — Beginner Tutorial

1. Introduction to Bootstrap

Bootstrap is a popular front-end framework for building responsive websites quickly.

  • Developed by Twitter engineers.
  • Includes pre-designed components like buttons, navbars, forms, modals.
  • Responsive grid system to adapt layouts to any screen size.

2. Setup & Installation

Add Bootstrap via CDN or install via npm:

                // CDN
                <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.5.0/dist/css/bootstrap.min.css" rel="stylesheet">
                <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.5.0/dist/js/bootstrap.bundle.min.js"></script>

                // NPM
                npm install bootstrap

3. Grid System

Bootstrap uses a 12-column responsive grid:

                <div class="container">
                  <div class="row">
                    <div class="col-md-6">Left</div>
                    <div class="col-md-6">Right</div>
                  </div>
                </div>

Columns automatically stack on smaller screens.

4. Components

Bootstrap provides ready-made UI components:

                // Buttons
                <button class="btn btn-primary">Click Me</button>

                // Cards
                <div class="card" style="width: 18rem;">
                  <div class="card-body">
                    <h5 class="card-title">Card Title</h5>
                    <p class="card-text">Some text.</p>
                  </div>
                </div>

5. Utilities

Bootstrap provides utility classes for spacing, colors, text alignment, display, and more:

                <div class="text-center text-primary p-3">Centered Text</div>
                <div class="m-2 p-2 bg-light">Box with margin and padding</div>

6. Forms

                <form>
                  <div class="mb-3">
                    <label for="email" class="form-label">Email address</label>
                    <input type="email" class="form-control" id="email">
                  </div>
                  <button type="submit" class="btn btn-success">Submit</button>
                </form>

Bootstrap forms are responsive and styled by default.

7. Navbar

                <nav class="navbar navbar-expand-lg navbar-dark bg-dark">
                  <div class="container-fluid">
                    <a class="navbar-brand" href="#">Brand</a>
                    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav"></button>
                    <div class="collapse navbar-collapse" id="navbarNav">
                      <ul class="navbar-nav">
                        <li class="nav-item"><a class="nav-link active" href="#">Home</a></li>
                      </ul>
                    </div>
                  </div>
                </nav>

8. Modals

                <button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
                  Launch Modal
                </button>

                <div class="modal fade" id="exampleModal">
                  <div class="modal-dialog">
                    <div class="modal-content">
                      <div class="modal-header">
                        <h5 class="modal-title">Modal title</h5>
                        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
                      </div>
                      <div class="modal-body">Modal content here</div>
                      <div class="modal-footer">
                        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Close</button>
                      </div>
                    </div>
                  </div>
                </div>

9. Best Practices

  • Use container classes (<div class="container"> or <div class="container-fluid">) to align content.
  • Use the grid system for responsive layouts.
  • Keep custom CSS minimal; prefer Bootstrap utilities.
  • Use semantic HTML and accessible components.
  • Always test on multiple screen sizes.

CSS — Beginner to Intermediate Tutorial

1. Introduction to CSS

CSS (Cascading Style Sheets) is used to style HTML documents. It controls colors, fonts, layout, and overall presentation.

  • Separates content (HTML) from design (CSS).
  • Supports responsive design.
  • CSS3 adds advanced features like animations, transitions, and flex/grid layout.

2. CSS Syntax

                    selector {
                      property: value;
                    }

Example:

                    p {
                      color: cyan;
                      font-size: 16px;
                    }

3. Selectors

                    /* Element selector */
                    p { ... }

                    /* Class selector */
                    .my-class { ... }

                    /* ID selector */
                    #my-id { ... }

                    /* Descendant selector */
                    div p { ... }

                    /* Grouping */
                    h1, h2, h3 { ... }

4. Properties & Values

Properties define what to style, and values define how.

                    color: red;
                    background-color: #222;
                    width: 100px;
                    height: 50px;
                    border: 1px solid cyan;

5. Colors & Backgrounds

                    /* Named colors */
                    color: red;

                    /* HEX colors */
                    color: #00ffff;

                    /* RGB colors */
                    color: rgb(0,255,255);

                    /* Background images */
                    background-image: url("bg.jpg");
                    background-size: cover;

6. Text & Fonts

                    font-family: Arial, sans-serif;
                    font-size: 16px;
                    font-weight: bold;
                    text-align: center;
                    text-decoration: underline;
                    line-height: 1.5;

7. Box Model

                    /* Box model properties */
                    width, height
                    padding
                    border
                    margin

Example:

                    div {
                      width: 100px;
                      padding: 10px;
                      border: 2px solid cyan;
                      margin: 5px;
                    }

8. Layout & Positioning

                    /* Positioning */
                    static, relative, absolute, fixed, sticky

                    /* Display types */
                    block, inline, inline-block, none

9. Flexbox & Grid

                    /* Flexbox */
                    .container {
                      display: flex;
                      justify-content: center;
                      align-items: center;
                    }

                    /* Grid */
                    .grid {
                      display: grid;
                      grid-template-columns: 1fr 2fr;
                      gap: 10px;
                    }

10. Transitions & Animations

                    /* Transition */
                    button {
                      transition: background 0.3s;
                    }

                    /* Keyframe animation */
                    @keyframes neonGlow {
                      0% { box-shadow: 0 0 5px cyan; }
                      50% { box-shadow: 0 0 20px cyan; }
                      100% { box-shadow: 0 0 5px cyan; }
                    }
                    button {
                      animation: neonGlow 1s infinite;
                    }

11. Best Practices

  • Keep CSS modular and organized.
  • Use external stylesheets for maintainability.
  • Use meaningful class and ID names.
  • Prefer relative units (%, em, rem) for responsive design.
  • Validate CSS using W3C CSS Validator.

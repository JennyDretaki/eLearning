Python — Beginner to Intermediate Tutorial

1. Introduction

Python is a high-level, interpreted, and dynamically typed language. It's known for its readability, versatility, and massive ecosystem of libraries.

  • Created by Guido van Rossum (1991).
  • Excellent for data science, AI, web development, and automation.
  • Emphasizes code simplicity and readability.

2. Setup & Installation

                    # Windows / macOS / Linux:
                    python --version
                    pip install package_name

Use an IDE like VS Code, PyCharm, or Jupyter Notebook for productivity.

3. Basic Syntax

                    print("Hello, Python!")

                    # Indentation defines blocks:
                    if True:
                        print("Indented block")
  • Python uses indentation (4 spaces) instead of braces.
  • Comments start with #.

4. Variables & Data Types

                    x = 10            # int
                    pi = 3.14         # float
                    name = "Alice"    # str
                    is_valid = True   # bool

Variables are dynamically typed — no need to declare types explicitly.

6. Control Flow

                    if x > 0:
                        print("Positive")
                    elif x == 0:
                        print("Zero")
                    else:
                        print("Negative")

                    for i in range(5):
                        print(i)

                    while x > 0:
                        x -= 1

7. Functions

                    def greet(name):

                    print(greet("Alice"))

8. Lists, Tuples & Dictionaries

                    # Lists
                    nums = [1, 2, 3]

                    # Tuples
                    coords = (10, 20)

                    # Dictionaries
                    person = {"name": "Bob", "age": 25}

9. Object-Oriented Programming

                    class Animal:
                        def __init__(self, name):
                            self.name = name

                        def speak(self):
                            print(f"{self.name} makes a sound")

                    class Dog(Animal):
                        def speak(self):
                            print(f"{self.name} barks")

                    dog = Dog("Rex")
                    dog.speak()

10. File Handling

                    with open("data.txt", "w") as f:
                        f.write("Hello file!")

                    with open("data.txt", "r") as f:
                        content = f.read()
                        print(content)

11. Exceptions

                    try:

                    finally:
                        print("Done")

13. Best Practices

  • Follow PEP 8 style guidelines.
  • Use virtual environments (venv).
  • Write clean, modular, and documented code.

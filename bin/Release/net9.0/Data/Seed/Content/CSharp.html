C# — Complete Beginner Tutorial

1. Introduction to C#

C# (pronounced “C-Sharp”) is a modern, object-oriented programming language developed by Microsoft. It’s widely used for building Windows applications, web services, APIs, games (via Unity), and more.

C# runs on the .NET framework (or .NET Core / .NET 8), providing a powerful runtime and huge standard library.

  • Created by Microsoft in early 2000s.
  • Supports OOP: classes, inheritance, polymorphism, encapsulation.
  • Used in enterprise systems, Unity game development, and cloud APIs.

2. Setting Up the Environment

To write and run C# code, install:

  • Visual Studio (recommended full IDE) — includes compiler and debugger.
  • Visual Studio Code + C# extension for a lightweight option.
  • .NET SDK (https://dotnet.microsoft.com/download) — for command-line builds.

To verify your setup, open a terminal and run:

dotnet --version

To create and run a console app:

                    dotnet new console -o MyApp
                    cd MyApp
                    dotnet run

3. Basic Syntax

                    using System;

                    class Program
                    {
                        static void Main(string[] args)
                        {
                            Console.WriteLine("Hello, World!");
                        }
                    }

💡 Explanation:

  • using System; — imports the System namespace (for Console class).
  • class Program — defines a class (entry point).
  • Main() — main method, where execution starts.
  • Console.WriteLine() — prints text to console.

6. Conditional Statements

                    int age = 18;

                    if (age >= 18)
                    {
                        Console.WriteLine("You are an adult.");
                    }
                    else
                    {
                        Console.WriteLine("You are underage.");
                    }

Switch example:

                    int day = 3;
                    switch(day)
                    {
                        case 1: Console.WriteLine("Monday"); break;
                        case 2: Console.WriteLine("Tuesday"); break;
                        default: Console.WriteLine("Other day"); break;
                    }

7. Loops

For loop:

                    for (int i = 0; i < 5; i++)
                    {
                        Console.WriteLine($"Count: {i}");
                    }

While loop:

                    int x = 0;
                    while (x < 3)
                    {
                        Console.WriteLine(x);
                        x++;
                    }

8. Methods and Parameters

                    static void Greet(string name)
                    {
                        Console.WriteLine($"Hello, {name}!");
                    }

                    Greet("Alice");

Return values:

                    static int Add(int a, int b)
                    {
                        return a + b;
                    }

                    int result = Add(5, 7);
                    Console.WriteLine(result);

9. Object-Oriented Programming (OOP)

                    class Person
                    {
                        public string Name;
                        public int Age;

                        public void Speak()
                        {
                            Console.WriteLine($"Hi, I'm {Name} and I'm {Age} years old.");
                        }
                    }

                    Person p = new Person();
                    p.Name = "Alice";
                    p.Age = 25;
                    p.Speak();

Includes four key principles:

  • Encapsulation: Protecting data inside classes.
  • Inheritance: Derive new classes from existing ones.
  • Polymorphism: Override or overload methods for flexibility.
  • Abstraction: Hide unnecessary details.

10. Arrays and Collections

Arrays:

                    int[] numbers = {1, 2, 3, 4};
                    Console.WriteLine(numbers[0]);

Lists:

                    using System.Collections.Generic;

                    List<string> names = new List<string>();
                    names.Add("Alice");
                    names.Add("Bob");

                    foreach(string name in names)
                    {
                        Console.WriteLine(name);
                    }

11. Error Handling

                    try
                    {
                        int result = 10 / 0;
                    }
                    catch (DivideByZeroException ex)
                    {
                        Console.WriteLine("Error: " + ex.Message);
                    }
                    finally
                    {
                        Console.WriteLine("Execution complete.");
                    }

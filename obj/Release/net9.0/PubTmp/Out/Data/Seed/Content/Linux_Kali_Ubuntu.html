Kali Linux & Ubuntu — Beginner to Intermediate Tutorial

1. Introduction to Linux

Linux is an open-source operating system based on the Unix architecture. It is known for its stability, flexibility, and strong community support.

  • Open-source and customizable.
  • Used for servers, cybersecurity, IoT, and software development.
  • Comes in many distributions (distros) — Kali Linux and Ubuntu are two popular examples.

2. Kali vs Ubuntu

  • Kali Linux: Designed for ethical hacking, penetration testing, and digital forensics. Comes preloaded with security tools.
  • Ubuntu: General-purpose distribution suitable for everyday use, development, and servers.
  • Base: Both are based on Debian, but differ in target audience and default configurations.

3. Installation & Setup

Both distros can be installed on bare metal, virtual machines, or dual-boot setups.

                    # Ubuntu installation (CLI)
                    sudo apt update && sudo apt upgrade -y

                    # Kali installation essentials
                    sudo apt install kali-linux-core -y

Use VirtualBox or VMware for sandboxed environments when testing tools or learning.

4. Essential Linux Commands

                    # File navigation
                    ls       # list files
                    cd dir   # change directory
                    pwd      # print working directory

                    # File operations
                    cp file1 file2
                    mv old new
                    rm file

                    # System info
                    uname -a
                    top
                    df -h

5. User & Permission Management

                    # Add a new user
                    sudo adduser username

                    # Switch user
                    su username

                    # Change permissions
                    chmod 755 file
                    chown user:group file

In Kali, many tools require root privileges — in Ubuntu, use sudo for security.

6. Package Management (APT)

APT (Advanced Package Tool) manages software installation in both Ubuntu and Kali.

                    sudo apt update
                    sudo apt install nmap
                    sudo apt remove package-name
                    sudo apt autoremove

Keep systems updated regularly to patch vulnerabilities and maintain stability.

7. Networking Tools

  • ping — test connectivity
  • ifconfig / ip a — view IP addresses
  • netstat / ss — show active network connections
  • nmap — scan open ports (Kali comes with this preinstalled)

8. Kali Linux Security Tools

  • Nmap: Network scanning
  • Wireshark: Packet analysis
  • Metasploit: Exploitation framework
  • Burp Suite: Web vulnerability testing
  • Hydra: Password brute-forcing

These tools are used in ethical hacking and cybersecurity research — always use them legally.

9. Ubuntu Software & Use Cases

  • Office & productivity: LibreOffice, Thunderbird, GIMP
  • Development: VS Code, Git, Docker
  • Server administration: Apache, Nginx, MySQL

Ubuntu focuses on usability, stability, and compatibility with cloud and enterprise systems.

10. System Customization

  • Change themes with GNOME Tweaks.
  • Edit startup services with systemctl.
  • Use aliases for faster terminal commands.
                    alias ll='ls -la'
                    alias gs='git status'

11. Linux Security & Updates

  • Use ufw firewall: sudo ufw enable
  • Install fail2ban to protect SSH from brute-force attacks.
  • Update regularly with sudo apt upgrade.
  • Encrypt disks and backups for data safety.

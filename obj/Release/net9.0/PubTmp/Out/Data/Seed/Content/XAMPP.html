XAMPP — Beginner Tutorial

1. What is XAMPP?

XAMPP is a free, open-source software package that provides a local web server environment for developing and testing PHP applications. It bundles:

  • X — Cross-platform (Windows, macOS, Linux)
  • A — Apache (web server)
  • M — MySQL/MariaDB (database)
  • P — PHP (scripting language)
  • P — Perl (optional scripting language)

2. Installing XAMPP

Follow these steps to install XAMPP:

                    1. Visit https://www.apachefriends.org/download.html
                    2. Download the correct installer for your OS.
                    3. Run the installer and follow setup steps.
                    4. Choose components: Apache, MySQL, PHPMyAdmin (essential ones).
                    5. Finish and launch the XAMPP Control Panel.

3. Components Overview

Main components of XAMPP include:

  • Apache: The local web server that handles HTTP requests.
  • MySQL/MariaDB: Database management system for storing data.
  • PHP: Server-side scripting language used for backend logic.
  • phpMyAdmin: Web interface for managing databases visually.
  • FileZilla: Optional FTP server for file transfers.

4. Setting Up Localhost

To start using XAMPP:

                    1. Open XAMPP Control Panel.
                    2. Click “Start” for Apache and MySQL.
                    3. Visit http://localhost in your browser.

If you see the XAMPP Dashboard, your server is working!

Project files go in the htdocs folder, typically found at:

C:\xampp\htdocs\

5. Using phpMyAdmin

phpMyAdmin helps you manage databases easily:

                    1. Go to http://localhost/phpmyadmin
                    2. Click “New” to create a new database.
                    3. Add tables, fields, and records visually.
                    4. Use the SQL tab for manual queries.

6. Creating a PHP Project

Let’s create a simple “Hello World” project:

                    1. Navigate to C:\xampp\htdocs
                    2. Create a folder named mysite
                    3. Inside it, create a file index.php with the following code:
                
                    <?php
                      echo "Hello, world! Welcome to XAMPP.";
                    ?>

Then, open your browser and visit:

http://localhost/mysite/

You should see your message displayed!

7. Troubleshooting

  • Apache not starting? Check if Skype, IIS, or another service is using port 80 or 443.
  • MySQL errors? Delete the `ib_logfile0` and `ib_logfile1` files if database won’t start.
  • Port conflict? Change Apache ports in XAMPP → Config → Service and Port Settings.
  • Permission issues (macOS/Linux)? Run XAMPP as administrator/sudo.

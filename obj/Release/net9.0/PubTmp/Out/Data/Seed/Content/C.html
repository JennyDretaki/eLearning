C — Detailed Beginner to Intermediate Tutorial

1. Introduction to C

C is a general-purpose, procedural programming language developed by Dennis Ritchie at Bell Labs in the 1970s.

  • Foundation for modern languages like C++, C#, and Java.
  • Compiled for speed and efficiency.
  • Supports structured and modular programming.

2. Setup & Tools

                    // Windows: Visual Studio or MinGW
                    // Linux: sudo apt install gcc
                    // macOS: brew install gcc

Compile and run:

gcc hello.c -o hello
                ./hello

Recommended IDEs: VS Code, Code::Blocks, CLion.

3. Basic Syntax

                    #include <stdio.h>

                    int main() {
                        printf("Hello, C!\n");
                        return 0;
                    }
  • Every C program starts with main().
  • Statements end with semicolons.
  • #include adds libraries.

8. Arrays & Strings

                    int arr[5] = {1,2,3,4,5};

9. Pointers

                    int x = 10;
                    int *ptr = &x;

                    printf("%d\n", *ptr); // 10

                    *ptr = 20;
                    printf("%d\n", x); // 20

11. File I/O

                    FILE *fp = fopen("data.txt", "w");
                    fprintf(fp, "Hello File\n");
                    fclose(fp);

                    fp = fopen("data.txt", "r");
                    char buffer[100];
                    fgets(buffer, 100, fp);
                    fclose(fp);

12. Dynamic Memory

                    int *arr = malloc(5 * sizeof(int));
                    if (arr != NULL) {
                        arr[0] = 10;
                        free(arr);
                    }

14. Best Practices

  • Initialize all variables before use.
  • Always check for NULL in file/memory operations.
  • Use descriptive names for functions and variables.
  • Keep code modular and well-commented.
